# site-de-mang-s
site
TML, CSS e JavaScript de um site de mangás básico (MangaVerse).

O código já possui:

Uma estrutura de página inicial (index.html) com cabeçalho, navegação e seções de conteúdo.

Um estilo (style.css) limpo e atraente com um tema de cores vibrantes (vermelho e branco) e um layout de grade responsivo para os cards de mangá.

Uma funcionalidade em JavaScript (script.js) que carrega dinamicamente uma lista de mangás pré-definidos (listaMangas) e cria os cards HTML, além de adicionar um evento de clique básico com um alert.
